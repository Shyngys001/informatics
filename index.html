Онлайн-платформа для подготовки к ҰБТ (Информатика)

Теоретический раздел

Здесь краткие конспекты по темам. Для полноты их нужно расширить и наполнить материалами.

1. Компьютерлік жүйелер

  • Компьютердің құрылғылары: Процессор, ОЗУ, HDD/SSD и т.д.
  • Компьютерлік желілер: Локальные и глобальные сети, топологии, протоколы.

2. Ақпараттық процестер

  • Ақпаратты ұсыну, кодтау: Биты, байты, ASCII, Unicode
  • Есептеу жүйелері: Двоичная, восьмеричная, шестнадцатеричная системы счисления.
  • Логические основы: Логические операции, булева алгебра.

3. Компьютерлік ойлау

  • Python программалау: Синтаксис, типы данных, функции, ООП.
  • Алгоритмдер: Функции, рекурсия, сортировки, графы.

4. Аппараттық және программалық қамтамасыз ету

Операционные системы, драйверы, взаимодействие «железо + софт».

5. Ақпараттық процестер мен жүйелер

  • Реляциондық деректер қоры: Таблицы, ключи, отношения.
  • Мәліметтер қорын жасау: Создание БД, SQL-запросы.

6. Web-жобалау

Основы HTML, CSS, JavaScript, адаптивная верстка, клиент-сервер.
